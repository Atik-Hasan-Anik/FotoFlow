# FotoFlow

A static photo gallery landing page built with HTML, CSS, and JavaScript.

## Features

- Responsive gallery with Pinterest-style cards
- Dark/light theme toggle with local storage support
- Infinite scroll loading for image cards
- Search filter for gallery items
- Like buttons with animated feedback
- Modal image preview with download and share actions
- Contact form for reaching the site owner

## Files

- `index.html` — main gallery page
- `about.html` — about page
- `contact.html` — contact page with form
- `style.css` — global layout and theme styling
- `script.js` — gallery behavior, theme logic, modal interactions
- `favicon.svg` — site favicon

## Usage

1. Open `index.html` in a browser.
2. Use the search bar to filter photos.
3. Click an image to open the modal.
4. Toggle theme with the button in the header.

## Notes

- The gallery uses placeholder images from Picsum.
- The contact page is set up for email-style submission.
- The project is designed to be a local static site without a backend.
